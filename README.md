# CareerAI

## AI-Based Career Readiness & Placement Intelligence Platform

### Description
CareerAI is an end-to-end data science project that predicts student placement chances, analyzes resumes, identifies skill gaps, and provides career insights using Machine Learning.

### Tech Stack
- Python
- Streamlit
- MySQL
- Scikit-Learn
- Pandas
- NumPy
- Plotly
- SQLAlchemy
- Git & GitHub
